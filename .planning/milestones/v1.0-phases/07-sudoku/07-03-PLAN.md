  <how-to-verify>
    1. Start local server: `cd /home/trashdev/projects/congames && python3 -m http.server 8080`
    2. Open Chrome at localhost:8080, enable DevTools device emulation: Custom 2000x1200, landscape, touch enabled
    3. Click "Sudoku" from welcome screen

    **Difficulty Selection:**
    4. Verify difficulty modal appears with 4 buttons: Maklik, Medium, Moeilik, Kenner (Afrikaans only, NO English subtitles)
    5. Select "Maklik" -- verify grid appears with ~45 filled cells
    6. Start new game, select "Kenner" -- verify grid appears with ~22 filled cells (may take a moment to generate)

    **Layout Check:**
    7. Verify 9x9 grid is visible with thick gold borders between 3x3 boxes
    8. Verify grid is square, fills available height, and doesn't overflow
    9. Verify number pad (3x3 grid of 1-9) is on the right side
    10. Verify action buttons (Vee Uit, Notas, Wenk, Kontroleer) below number pad
    11. Verify all cells and buttons meet 56px minimum tap target
    12. Verify header shows timer, undo, pause, and new game buttons

    **Input Flow (cell-first):**
    13. Tap an empty cell -- verify it highlights with gold border/glow
    14. Tap a number on the pad -- verify number appears in accent gold in the cell
    15. Tap a cell with a given number -- verify it's in white/bright and cannot be changed
    16. Try entering a number in a given cell -- should be refused

    **Number Highlighting:**
    17. Tap a cell containing a number -- verify ALL instances of that number highlight across the entire grid
    18. Verify cells in same row/col/box as selected cell get subtle background highlight

    **Notes Mode:**
    19. Tap "Notas" button -- verify visual indicator that notes mode is active
    20. Tap an empty cell, then tap numbers -- verify small pencil marks appear in 3x3 mini-grid
    21. Tap same number again -- verify pencil mark toggles off
    22. Toggle notes mode off, enter a number in a cell -- verify pencil marks in related cells auto-clear for that digit

    **Completed Number Greying:**
    23. Fill all 9 instances of a digit -- verify that digit becomes greyed out on the number pad

    **Hints and Error Checking:**
    24. Tap "Wenk" -- verify one empty cell fills with correct value and sound plays
    25. Enter a wrong number in a cell, then tap "Kontroleer" -- verify the wrong cell flashes red briefly (1-2 seconds) then fades back
    26. If no errors, tap "Kontroleer" -- verify toast says "Geen foute!"

    **Undo:**
    27. Enter a number, tap undo -- verify the cell returns to previous state
    28. Toggle a note, tap undo -- verify note toggles back

    **Timer and Pause:**
    29. Verify timer counts up in the header
    30. Tap pause button -- verify overlay covers grid showing elapsed time and "Hervat" button
    31. Tap "Hervat" -- verify game resumes with timer continuing

    **Auto-Save and Resume:**

    33. Navigate back to Sudoku -- verify "Wil jy voortgaan?" prompt appears
    34. Tap "Hervat" -- verify game restores with correct board state and timer
    35. Navigate back, return, tap "Nuwe Spel" -- verify difficulty modal appears for fresh game

    **Win Condition:**
    36. Complete a puzzle (Maklik is easiest) -- verify win celebration triggers with sound and overlay showing time
