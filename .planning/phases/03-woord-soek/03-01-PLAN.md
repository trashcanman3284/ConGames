  <name>Task 1: Create puzzle engine (games/woordsoek/engine.js)</name>
  <files>games/woordsoek/engine.js</files>
  <action>
Create `games/woordsoek/engine.js` as an IIFE exposing `WoordSoekEngine` globally. No DOM access, no imports/exports. Pure data in, data out.

**Constants:**
- `DIRECTIONS` object mapping 8 compass names to [rowDelta, colDelta] vectors: NORTH [-1,0], SOUTH [1,0], EAST [0,1], WEST [0,-1], NORTH_EAST [-1,1], NORTH_WEST [-1,-1], SOUTH_EAST [1,1], SOUTH_WEST [1,-1]
- `DIFFICULTY` config object with three levels per user decision:
  - `easy`: { rows: 10, cols: 10, wordCount: 8, minLen: 4, maxLen: 7, label: 'Maklik' }
  - `medium`: { rows: 12, cols: 12, wordCount: 12, minLen: 4, maxLen: 9, label: 'Medium' }
  - `hard`: { rows: 15, cols: 15, wordCount: 18, minLen: 4, maxLen: 12, label: 'Moeilik' }
- `FILL_LETTERS` string weighted towards common Afrikaans letters: `'AAABCDDEEEEFGGHIIIJKKLLMNNOOOPRRSSSSTTTTUUVWYZ'` (no Q — absent from dataset)
- `HIGHLIGHT_COLOURS` array of 8 rgba colours visible on warm dark background:
  `['rgba(91,155,213,0.45)', 'rgba(76,175,120,0.45)', 'rgba(224,150,60,0.45)', 'rgba(180,100,200,0.45)', 'rgba(230,80,80,0.45)', 'rgba(60,200,200,0.45)', 'rgba(220,200,60,0.45)', 'rgba(200,120,160,0.45)']`

**Functions:**

`filterWords(allWords)` — Takes the raw words.json array. Returns only words that are purely alphabetic (`/^[a-zA-Z]+$/`), length >= 4. Converts all to uppercase. Caches result.

`generatePuzzle(filteredWords, difficulty)` — Takes filtered word array + difficulty key ('easy'/'medium'/'hard'). Returns `{ grid: string[][], placedWords: [{ word, row, col, dirName, cells: [{r,c}] }], rows, cols }`.
  - Get config from DIFFICULTY[difficulty]
  - Filter words by length range (minLen..maxLen)
  - Shuffle filtered words using Fisher-Yates
  - Create empty grid (null-filled 2D array)
  - Attempt to place wordCount words: for each word, try up to 100 random position+direction combos. Use `canPlace()` to check collisions (null cell or matching letter). If placement fails after 100 tries, skip word, try next from pool. Keep a pool 3x larger than wordCount to ensure enough words are placed.
  - Fill remaining null cells with `randomFillLetter()` from FILL_LETTERS
  - Return puzzle object. If fewer than wordCount words placed, log warning but return what was placed (minimum 5 words or retry entire generation, up to 3 retries).

`canPlace(grid, word, startR, startC, dr, dc, rows, cols)` — Check if word fits at position in direction without conflicting collisions. Return boolean.

`doPlace(grid, word, startR, startC, dr, dc)` — Write word letters into grid cells. Return cells array [{r,c}].

`checkSelection(startR, startC, endR, endC, placedWords)` — Validate a tap-tap selection:
  1. Calculate dr = endR - startR, dc = endC - startC
  2. Valid direction requires: dr===0 OR dc===0 OR abs(dr)===abs(dc). If not, return { found: false }
  3. Normalize to unit direction: Math.sign(dr), Math.sign(dc)
  4. Calculate length = Math.max(abs(dr), abs(dc)) + 1
  5. Check all placedWords: match if word.row === startR, word.col === startC, word direction matches, and word.length === length. Also check REVERSE: word's last cell matches start, first cell matches end.
  6. Return { found: true, word: string, index: number } or { found: false }

`getDirection(r1, c1, r2, c2)` — Returns { dr, dc, length } if valid 8-direction line, or null if not. Used by UI for preview line.

`getCellsInLine(r1, c1, r2, c2)` — Returns array of {r, c} for all cells from (r1,c1) to (r2,c2) along a valid 8-direction line. Returns empty array if invalid direction.

`shuffle(arr)` — Fisher-Yates in-place shuffle. Returns arr.

`randomFillLetter()` — Pick random character from FILL_LETTERS.

**Expose:** `return { generatePuzzle, checkSelection, getDirection, getCellsInLine, filterWords, DIRECTIONS, DIFFICULTY, HIGHLIGHT_COLOURS }`
