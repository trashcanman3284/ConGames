**css/cards.css** — Create the complete card stylesheet. Follow the research doc's "Complete Card CSS" example closely, with these specifics:

1. **Card variables** (add to `:root`):
   - `--card-red: #c0392b` (warm red, readable on cream)
   - `--card-black: #1a1a1a` (near-black)
   - `--card-bg: #f5f0e8` (off-white/cream)
   - `--card-border: #d0c8b8`
   - `--card-shadow: 0 2px 6px rgba(0,0,0,0.3)`
   - `--card-back-bg: #2c5f8a` (warm dark blue)
   - `--card-back-pattern: #1e4568` (darker blue for stripes)
   - `--card-back-border: #4a7fa8`
   - `--card-radius: 8px`

2. **Base `.card`**: `width: 100%; aspect-ratio: 5/7; position: relative; border-radius: var(--card-radius); box-shadow: var(--card-shadow); overflow: hidden; font-family: var(--font-body); user-select: none; -webkit-user-select: none;`

3. **Face-up** (`.card.face-up`): cream background, subtle border.

4. **Suit colours**: `.card.suit-hearts, .card.suit-diamonds` → `color: var(--card-red)`. `.card.suit-spades, .card.suit-clubs` → `color: var(--card-black)`.

5. **Corner labels** (`.card-corner`): absolute positioned, flex column, `font-weight: 700`. Use `font-size: clamp(0.7rem, 1.2vw, 1.1rem)` (NOT `cqw` — use `vw` for Android 10 compatibility per research). `.card-corner.top-left` at top:4px left:5px. `.card-corner.bottom-right` at bottom:4px right:5px with `transform: rotate(180deg)`.

6. **Suit symbol** (`.suit-symbol`): `font-size: 0.85em; line-height: 1;`

7. **Center pip** (`.card-center`): absolute center, `font-size: clamp(1.5rem, 5vw, 3rem); line-height: 1;`

8. **Face-down** (`.card.face-down`): blue background with `repeating-linear-gradient(45deg, transparent, transparent 4px, var(--card-back-pattern) 4px, var(--card-back-pattern) 5px)`. Add `::after` pseudo-element for inner border (inset 4px, 1px solid, slightly rounded).

9. **Card stacking** (`.card-stack`): flex column. `.card-stack .card { margin-top: -75%; }` with `:first-child { margin-top: 0; }`. `.card-stack .card.face-down { margin-top: -82%; }`.

10. **Selected state** (`.card.selected`): Add a gold outline/border using `box-shadow: 0 0 0 3px var(--accent-gold, #d4a23a)` for when games highlight a tapped card. This is a convenience class games will use.

11. **Placeholder** (`.card-placeholder`): Empty card slot. Same dimensions as card, dashed border, transparent background. Used for empty foundations/free cells.
