  <action>
Present the complete Solitaire game for human verification. Start the dev server and guide the user through testing.

What was built: Complete Klondike Solitaire game with tap-to-move, drag-and-drop, undo, auto-complete, Windows-style win animation, draw mode selection, and settings toggles.

How to verify:
1. Start server: `python3 -m http.server 8080` from project root
2. Open Chrome DevTools, set device to 2000x1200 landscape with touch enabled
3. Navigate to `http://localhost:8080`, tap "Solitaire" game button

**Deal and layout:**
4. Draw mode modal should appear — select "Trek 1 kaart"
5. Verify: 7 tableau columns (1-7 cards, top cards face-up), stock pile (face-down), 4 empty foundation placeholders

**Basic moves:**
6. Tap a face-up card — should get gold glow selection
7. Tap a valid tableau column — card should move there
8. Try an invalid move (wrong colour/rank) — should not work
9. Move an Ace to an empty foundation — should accept
10. Tap undo — last move should reverse

**Stock and waste:**
11. Tap stock pile — one card should flip to waste
12. Tap the waste card — select it, move to tableau or foundation
13. Keep tapping stock until empty — tap empty stock to recycle waste

**Drag interaction:**
14. Touch and drag a face-up card to a valid column — should move on drop
15. Drag a stack of cards (valid sequence) — entire stack should move

**Settings:**
16. Tap settings gear icon — modal should open
17. Toggle timer on/off — timer in header should appear/disappear
18. Toggle destination highlighting on — select a card, valid targets should glow gold
19. Close settings modal

**Win flow (if able to win, or trust auto-complete):**
20. Play until all tableau cards are face-up — auto-complete should trigger
21. Cards should animate one-by-one to foundations
22. On win: cascading bouncing cards animation should play (Windows-style with trail)
23. Win overlay should show time, moves, and "Nuwe Spel" / "Tuis" buttons

**Edge cases:**
24. Tap "Nuwe Spel" mid-game — should start fresh (no confirmation)
25. Tap "Terug" — should return to welcome screen
26. Re-enter Solitaire — draw mode modal or new game should appear
27. Check no JS errors in console throughout
