**DIFFICULTY constant:**
```javascript
var DIFFICULTY = {
  maklik:  { label: 'Maklik',  size: 9,  wordCount: 7  },
  medium:  { label: 'Medium',  size: 13, wordCount: 13 },
  moeilik: { label: 'Moeilik', size: 17, wordCount: 18 }
};
```

**_state object:**
```javascript
var _state = {
  grid:           [],   // 2D array of {letter, number, isBlack, entered}
  words:          [],   // [{word, clue, number, direction, row, col, length, complete}]
  difficulty:     '',
  startTime:      0,
  elapsedSeconds: 0,
  undoStack:      []
};
```

**Internal helper functions to implement:**

1. `shuffle(arr)` — Fisher-Yates, copy from SudokuEngine verbatim (returns new array, does not mutate).

2. `createEmptyGrid(size)` — Returns size x size 2D array where every cell is `{ letter: '', number: 0, isBlack: true, entered: '' }`. All cells start as black; placement turns them white.

3. `deepCloneState()` — Returns deep clone of `{grid, words, difficulty, elapsedSeconds}` for getState(). Clone grid cell-by-cell (object spread or manual copy), clone words array with object copies.

4. `selectWords(clues, config)` — Bucket selection per locked decision:
   - Bucket clues into short (4-5 chars), medium (6-7 chars), long (8-10 chars)
   - Shuffle each bucket
   - Target ratio: 30% short, 40% medium, 30% long of `config.wordCount`
   - Round: `shortCount = Math.round(config.wordCount * 0.3)`, `medCount = Math.round(config.wordCount * 0.4)`, `longCount = config.wordCount - shortCount - medCount`
   - If a bucket has fewer than needed, take what's available, fill remainder from other buckets
   - For anchor selection: always pick the longest word from the long bucket (min length 8 for moeilik). If no 8+ char words, take longest available.
   - Sort selected words by length descending (anchor first)
   - Return array of `{word, clue}` objects

5. `placeAnchor(grid, wordObj, size)` — Place first (longest) word horizontally at grid centre:
   - `midRow = Math.floor(size / 2)`
   - `startCol = Math.floor((size - wordObj.word.length) / 2)`
   - Set each cell: `grid[midRow][startCol + c].letter = wordObj.word[c]`, `grid[midRow][startCol + c].isBlack = false`
   - Return word entry: `{word: wordObj.word, clue: wordObj.clue, number: 0, direction: 'across', row: midRow, col: startCol, length: wordObj.word.length, complete: false}`

6. `canPlace(grid, word, row, col, direction, size)` — Check if word fits at position:
   - `dr = direction === 'down' ? 1 : 0`, `dc = direction === 'across' ? 1 : 0`
   - For each letter index i: check bounds (`r >= 0 && r < size && c >= 0 && c < size`)
   - Cell must be either black (will be converted to white) OR already have the same letter (intersection)
   - Return false if cell has a different letter

7. `checkEndpointClearance(grid, row, col, direction, length, size)` — Per locked decision, blank/edge before first letter and after last:
   - Before: `preR = row - dr`, `preC = col - dc`. If in bounds and `grid[preR][preC].letter !== ''`, return false.
   - After: `endR = row + dr * length`, `endC = col + dc * length`. If in bounds and `grid[endR][endC].letter !== ''`, return false.
   - Return true otherwise.

8. `checkParallelAdjacency(grid, word, row, col, direction, size, placedWords)` — Prevent side-by-side parallel words:
   - For each cell the candidate would occupy at `(r, c)`:
     - Check perpendicular neighbours (both sides)
     - `pr = direction === 'across' ? 1 : 0` (row offset for perpendicular)
     - `pc = direction === 'down' ? 1 : 0` (col offset for perpendicular)
     - For side in [-1, 1]: `nr = r + side * pr`, `nc = c + side * pc`
     - If neighbour cell has a letter AND that cell is NOT the intersection point (i.e., the candidate word's letter at position i does not match the cell letter at (r,c) — meaning this is not a crossing), check if the neighbour is part of a word running in the SAME direction as the candidate. If yes, reject.
   - Implementation of `isPartOfWordInDirection(placedWords, nr, nc, direction)`: iterate placed words of same direction, check if (nr, nc) falls within their span.

9. `findCandidatePlacements(word, placedWords, grid, size, startTime)` — Core intersection search:
   - Check `Date.now() - startTime >= 2000` at the start — return empty array if time exceeded (Pitfall 4)
   - For each placed word, for each letter position in that placed word:
     - For each letter position in the candidate word:
       - If letters match: compute candidate placement position
       - Candidate direction is opposite of placed word's direction
       - Compute startRow/startCol from the intersection point
       - Run `canPlace()`, `checkEndpointClearance()`, `checkParallelAdjacency()`
       - If all pass: count total intersections with ALL placed words (not just this one) for scoring
       - Push `{row, col, direction, intersections}` to candidates
   - Return candidates array

10. `scoreCandidate(candidate, size)` — Prefer more intersections and central placement:
    - `score = (candidate.intersections * 10) - (Math.abs(candidate.row - size/2) + Math.abs(candidate.col - size/2))`

11. `countIntersections(grid, word, row, col, direction)` — Count how many cells in the word's span already have the same letter placed (shared with another word):
    - Walk the word's cells, count cells where `grid[r][c].letter === word[i]` and `!grid[r][c].isBlack`

12. `commitPlacement(grid, wordObj, row, col, direction)` — Place word onto grid:
    - Set each cell's `letter` and `isBlack = false`
    - Return word entry object with `complete: false`

13. `assignCellNumbers(grid, words, size)` — Left-to-right, top-to-bottom numbering (ENG-04):
    - `num = 1`
    - Scan row-major: for r 0..size-1, for c 0..size-1:
      - Skip black cells
      - `startsAcross`: (c === 0 OR grid[r][c-1].isBlack) AND (c + 1 < size AND !grid[r][c+1].isBlack)
      - `startsDown`: (r === 0 OR grid[r-1][c].isBlack) AND (r + 1 < size AND !grid[r+1][c].isBlack)
      - Also check: a single-cell word is invalid — startsAcross requires at least 2 consecutive non-black cells, startsDown requires at least 2 consecutive non-black rows
      - If startsAcross or startsDown: `grid[r][c].number = num`, then find matching word(s) in `words` array by (row, col, direction) and set their `.number = num`, increment num
    - One number per cell even if it starts both an Across and Down word

14. `_tryGenerate(config, clues, startTime)` — Single generation attempt:
    - Call `selectWords(clues, config)` to get word list
    - `createEmptyGrid(config.size)`
    - `placeAnchor()` with first (longest) word
    - Loop remaining words in descending length order:
      - `findCandidatePlacements()` — check time cap inside
      - If candidates exist: pick highest-scored via `scoreCandidate()`, `commitPlacement()`
    - Return `{grid, words: placedWords}`

15. `generate(difficulty, clues)` — Public entry point (ENG-05 retry logic):
    - Validate difficulty key exists in DIFFICULTY
    - `var config = DIFFICULTY[difficulty]`
    - `var startTime = Date.now()`
    - `var bestResult = null`
    - Loop up to 5 attempts:
      - Break if `Date.now() - startTime >= 2000`
      - `var result = _tryGenerate(config, clues, startTime)`
      - Track bestResult (highest `result.words.length`)
      - Break early if `result.words.length >= config.wordCount`
    - `assignCellNumbers(bestResult.grid, bestResult.words, config.size)`
    - Reset `_state`: grid, words, difficulty, startTime = Date.now(), undoStack = [], elapsedSeconds = 0
    - Return `getState()` (deep clone)

**Game state API functions:**

16. `setLetter(row, col, letter)` — Set entered letter:
    - Validate: row/col in bounds, cell is not black
    - Push `{row, col, prevEntered: _state.grid[row][col].entered}` to `_state.undoStack`
    - Set `_state.grid[row][col].entered = letter.toUpperCase()`

17. `checkWord(number, direction)` — Check if entered letters match:
    - Find word in `_state.words` by number + direction
    - Walk cells, compare `entered.toUpperCase()` to `letter`
    - Return `{correct: boolean, letters: [{row, col, correct}]}`
    - If all correct, set `word.complete = true`

18. `isComplete()` — Return `_state.words.every(w => w.complete)` equivalent (use for loop for ES5 compat, or `.every()` is fine — supported in Android 4.4+)

19. `undo()` — Pop undoStack, revert `entered` to `prevEntered`. Return `{success: true}` or `{success: false}` if stack empty.

20. `getState()` — Return `deepCloneState()`.

21. `getElapsed()` — Return `Math.floor((Date.now() - _state.startTime) / 1000)`.

**Return object:**
```javascript
return {
  DIFFICULTY: DIFFICULTY,
  generate:   generate,
  setLetter:  setLetter,
  checkWord:  checkWord,
  isComplete: isComplete,
  undo:       undo,
  getState:   getState,
  getElapsed: getElapsed
};
```
